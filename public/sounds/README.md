# Sonidos de foco (opcional)

Los sonidos "generados" (ruido marrón, rosa, blanco, lluvia, viento, olas) funcionan
solos, sin archivos y sin internet. No hace falta hacer nada para que anden.

Si además querés **Lofi** o **Cafetería**, tenés que poner acá un archivo de audio:

| Sonido en la app | Archivo que va acá |
| ---------------- | ------------------ |
| Lofi             | `lofi.mp3`         |
| Cafetería        | `cafe.mp3`         |

## Cómo agregarlos

1. Bajá un loop **libre de regalías** (sin derechos de autor). Buenos lugares gratis:
   - Pixabay Music (https://pixabay.com/music/) — filtrá por "loop"
   - Free Music Archive (https://freemusicarchive.org/) — buscá licencias CC0
2. Renombrá el archivo exactamente a `lofi.mp3` o `cafe.mp3`.
3. Ponelo en esta carpeta (`public/sounds/`).
4. Volvé a publicar la app. El sonido aparece desbloqueado automáticamente.

> Ojo: no subas música con derechos de autor (Spotify, YouTube, etc.). Para eso está
> la conexión con tu cuenta de Spotify.
